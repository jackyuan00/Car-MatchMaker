# **App Name**: Mercedes Benz AI Configurator

## Core Features:

- JSON Question Loading: Load questions from a JSON file to populate the quiz.
- Question Answering: Allow users to answer multiple-choice questions.
- AI Recommendation Tool: Based on the user's answers, the AI tool recommends a Mercedes-Benz model that fits their preferences using a reasoning chain.
- Car Presentation: Presents the configures car in appealing UI

## Style Guidelines:

- Primary color: Deep blue (#1A237E) evoking luxury and sophistication, reflective of the Mercedes-Benz brand.
- Background color: Light gray (#ECEFF1), providing a clean and modern backdrop.
- Accent color: Analogous teal (#00BCD4), contrasting against the deep blue to highlight interactive elements and call to actions.
- Headline font: 'Poppins', a geometric sans-serif for a precise and contemporary look.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use minimalist icons that represent different car features and user preferences.
- Modern, card-based design with plenty of whitespace to keep the focus on the questions and car recommendations.
- Subtle animations for transitions between questions and revealing the recommended car.
